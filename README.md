# NutriVet

NutriVet is a veterinary nutrition calculator for dogs and cats.

## Features
- Species selection (Dog/Cat)
- Breed selection
- Age and weight based calculations
- RER and MER calculation
- NRC nutrition recommendations
- Protein, Fat, Carbohydrate and Water requirements
- Health-based dietary recommendations
- Multiple diet options

## Technologies Used
- HTML
- CSS
- JavaScript
